Boilerplate NextJS + GRaphQL

using in boilerplate:

- NextJS
- GraphQL
- Codegen
- SASS

## Getting Started

First, create application:

```bash

npx nextjs-graphql project-name

```

## Initializing

Install dependencies:

```bash
yarn
#or
npm install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```
